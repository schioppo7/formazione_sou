# Vagrant Ping Pong

Esercizio di configurazione e provisioning con Vagrant, pensato per creare una semplice dimostrazione di comunicazione tra macchine virtuali in stile "ping pong".

## Descrizione

L'obiettivo è usare Vagrant per avviare e configurare due macchine virtuali, con all'interno docker installato, per effettuare una migrazione di container tra una macchina all'altra.
